# متطلبات تطبيق إحصاء منتجات السوبرماركت

## متطلبات النظام
- Flutter SDK (الإصدار 3.0.0 أو أحدث)
- Dart SDK (الإصدار 3.0.0 أو أحدث)
- Android SDK (للتطوير لنظام Android)
- JDK (Java Development Kit) الإصدار 11 أو أحدث

## أدوات التطوير
- Android Studio أو Visual Studio Code
- Flutter و Dart plugins للـ IDE
- Git (للتحكم في الإصدارات)

## متطلبات الأجهزة
- جهاز Android بإصدار 5.0 (Lollipop) أو أحدث
- كاميرا خلفية لمسح الباركود
- مساحة تخزين كافية (50MB على الأقل)

## تعليمات التثبيت
1. تثبيت Flutter SDK من الموقع الرسمي: https://flutter.dev/docs/get-started/install
2. تثبيت Android Studio أو Visual Studio Code
3. تثبيت Flutter و Dart plugins للـ IDE
4. تنفيذ الأمر `flutter doctor` للتأكد من إعداد البيئة بشكل صحيح
5. تنفيذ الأمر `flutter pub get` لتثبيت جميع الاعتمادات

## ملاحظات إضافية
- يجب تفعيل وضع المطور على جهاز Android للتثبيت المباشر
- للتصدير إلى ملف APK، استخدم الأمر: `flutter build apk --release`
- للتشغيل المباشر على الجهاز، استخدم الأمر: `flutter run`
